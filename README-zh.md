## 核心功能

1. 🧬 **指纹浏览器**  
   为每个账号创建独立浏览器环境，实现指纹级隔离，降低账号关联与风控风险。

2. 🌍 **IP / 代理支持**  
   支持 HTTP / HTTPS / SOCKS5 等代理类型，灵活绑定账号与网络环境。

3. 🤖 **自动化脚本能力**  
   支持自动化脚本执行，实现 Twitter（X）、Discord 等平台的自动登录与操作。

4. 🧩 **浏览器插件支持**  
   支持安装并管理常用浏览器插件，满足多样化业务与运营需求。

5. 👥 **多账号统一管理**  
   集中管理 Twitter、Telegram、Discord、Email 等主流平台账号，支持批量操作。

6. 💻 **多平台兼容**  
   支持 Windows / macOS 系统，稳定运行，适配不同工作环境。
